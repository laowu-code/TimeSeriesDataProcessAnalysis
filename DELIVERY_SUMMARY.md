## ✨ 已实现的功能

### ✅ 功能模块 1: 📥 数据交互 (Data I/O)
- [x] 上传 CSV/Excel 文件
- [x] 自动识别时间列和数值列
- [x] 即时数据预览和元数据展示
- [x] 导出清洗数据为 CSV/Excel

### ✅ 功能模块 2: 📋 数据概览与统计 (EDA)
- [x] 实时数据预览（前 10 行）
- [x] 统计描述（均值、中位数、标准差、方差、偏度、峰度）
- [x] 缺失值分析（数值 + 可视化）

### ✅ 功能模块 3: 🔍 时间完整性检查与插补 (Data Cleaning)
- [x] 时间频率检测（自动识别主频率）
- [x] 断点识别和统计
- [x] 6 种插补算法：
  - Linear（线性插补）
  - Polynomial（多项式）
  - Spline（样条曲线）
  - Mean（均值填充）
  - FFill（前向填充）
  - BFill（后向填充）
- [x] 插补效果对比可视化

### ✅ 功能模块 4: 📈 动态可视化 (Visualization)
- [x] 3 种图表类型（折线图、区域图、柱状图）
- [x] Plotly 交互式图表（缩放、平移、数值查看）
- [x] 多变量同时显示
- [x] 移动平均（MA）预览

### ✅ 功能模块 5: ⚠️ 异常检测与处理 (Outlier Detection)
- [x] 3 种检测算法：
  - 3σ 原则（Z-Score）
  - IQR（四分位距）
  - 滚动窗口检测
- [x] 异常点可视化标记
- [x] 2 种处理方式（删除或插补）

### ✅ 功能模块 6: 🔬 高级分析扩展 (Advanced Analysis)
- [x] STL 分解（趋势+季节性+残留）
- [x] 相关性分析（热力图 + Top10 相关对）
- [x] 多列对比（支持归一化）



```bash

streamlit run app.py
```

## 📞 故障排查速查表

| 问题 | 解决方案 |
|------|--------|
| ModuleNotFoundError | `pip install -r requirements.txt` |
| 时间列未识别 | 确保时间格式标准（YYYY-MM-DD） |
| Streamlit 很慢 | 首次启动正常，之后会快速响应 |
| 插补后仍有 NaN | 尝试 `mean` 或 `ffill` 方法 |
| 上传文件过大 | 编辑 `.streamlit/config.toml` 增加限制 |

**更多问题**：查看 README.md 或 QUICKSTART.md
