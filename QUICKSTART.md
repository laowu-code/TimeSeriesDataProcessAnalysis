# 🚀 快速入门指南

## 60 秒快速开始

### Windows 用户 (最简单)

1. **双击** `run.bat` 文件
2. 耐心等待依赖安装（第一次需要 2-3 分钟）
3. 浏览器自动打开，开始使用！

### macOS / Linux 用户

```bash
# 1. 进入项目目录
cd DataPrecess

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动应用
streamlit run app.py
```

或直接运行：
```bash
python run.py
```

## 📖 基本操作步骤

### 步骤 1️⃣：导入数据 (📥 数据导入)

```
侧边栏 → 选择 "📥 数据导入" → 点击上传区域 → 选择你的 CSV 或 Excel 文件
```

✅ 应用会自动：
- 识别时间列（如 timestamp、date、time）
- 识别数值列（如 temperature、pm25、price）
- 显示数据摘要

**样本数据**：使用提供的 `sample_data.csv` 进行测试

---

### 步骤 2️⃣：查看数据统计 (📋 数据概览)

```
侧边栏 → 选择 "📋 数据概览"
```

您将看到：
- 📊 数据预览（前 10 行）
- 📈 统计描述（均值、中位数、标准差等）
- 🔎 缺失值分析及可视化

---

### 步骤 3️⃣：清洗时间数据 (🔍 数据清洗)

```
侧边栏 → 选择 "🔍 数据清洗"
```

操作：
1. 查看 **时间连续性检查** → 了解数据中的时间断点
2. 在 **智能插补配置** 中：
   - ✓ 选择需要填补的列
   - ✓ 选择插补方法（推荐：linear）
   - 🔄 点击 **执行插补**
3. 📊 在下方查看原始数据与插补后的对比

**推荐方案**：
- 短期缺失（< 5%）→ **linear**
- 平滑变化 → **polynomial**
- 快速填补 → **ffill** 或 **bfill**

---

### 步骤 4️⃣：可视化分析 (📈 可视化分析)

```
侧边栏 → 选择 "📈 可视化分析"
```

功能：
1. **选择要显示的列** → 勾选多个变量
2. **选择图表类型** → 折线图、区域图、柱状图
3. **启用移动平均** → 勾选显示 MA 曲线（平滑噪声）

💡 **交互功能**：
- 🖱️ 拖拽选区缩放
- 🔍 悬停查看具体数值
- 📌 双击重置视图

---

### 步骤 5️⃣：检测异常值 (⚠️ 异常检测)

```
侧边栏 → 选择 "⚠️ 异常检测"
```

操作：
1. **选择分析列** → 比如 "temperature"
2. **选择检测方法**：
   - **zscore** (3σ原则) → 最常用
   - **iqr** → 四分位距，对异常值稳健
   - **rolling** → 滚动窗口，捕捉局部异常
3. 🔍 **执行异常检测**

4. 查看结果：
   - 异常值用 🔴 红色/⬥钻石 标记
   - 选择处理方式：
     - ✂️ **删除异常行**
     - 🔧 **视为缺失并插补**

---

### 步骤 6️⃣：高级分析 (🔬 高级分析)

```
侧边栏 → 选择 "🔬 高级分析"
```

**3 种分析方法**：

#### 📉 季节性分解 (STL)
- 将时序分解为：趋势 + 季节性 + 噪声
- 适用于：具有周期性的数据（日/周/月）
- **参数**：季节周期（默认 12）

#### 🔗 相关性分析
- 显示所有变量间的关联系数热力图
- 列出强相关对 Top 10
- **颜色含义**：红色（正相关） ↔ 蓝色（负相关）

#### 📊 多列对比
- 同时显示 2+ 列的变化趋势
- ✓ 勾选 **归一化** 查看不同量级数据的变化模式

## 🎯 常见场景示例

### 📌 场景 1：修复缺失的传感器数据

```
1. 📥 上传包含 null 值的 CSV
2. 📋 查看缺失率 (数据概览 → 缺失值分析)
3. 🔍 执行线性插补 (数据清洗 → 智能插补)
4. 📈 验证效果 (对比原始与插补数据)
5. 💾 导出清洁数据
```

### 📌 场景 2：发现股票价格异常

```
1. 📥 上传股票日 K 线数据
2. 📈 绘制价格时间序列 (可视化分析)
3. ⚠️ 用 IQR 方法检测异常跳跃 (异常检测)
4. 🔬 分析相关性 (与成交量、波动率关联)
5. 📊 输出异常事件报告
```

### 📌 场景 3：识别温度季节性

```
1. 📥 导入 1 年的每日温度数据
2. 📋 查看季度统计 (数据概览)
3. 🔬 STL 分解（周期参数设为 365）
4. 观察：趋势 (升温中) vs 季节性 (冬夏差异)
5. 📈 预测来年温度范围
```

## ⚙️ 高级配置（可选）

### 增加上传文件大小限制

编辑 `~/.streamlit/config.toml`：

```toml
[server]
maxUploadSize = 500  # 单位：MB，默认 200
```

### 修改应用外观

编辑 `~/.streamlit/config.toml`：

```toml
[theme]
base = "light"  # 或 "dark"
primaryColor = "#FF6B6B"

# 也可在应用右上角 ☰ → Settings 中实时切换
```

## 🐛 常见问题排查

### ❌ 问题：ModuleNotFoundError

**原因**：依赖未安装

**解决**：
```bash
pip install -r requirements.txt
```

---

### ❌ 问题：时间列未被自动识别

**原因**：时间格式不标准

**解决**：
- ✓ 支持的格式：
  - `2024-01-01 10:30:00`
  - `2024/01/01 10:30`
  - `01/01/2024`
  - `2024-01-01`

- ❌ 不支持：`01-January-2024` 等文本格式

---

### ❌ 问题：插补后仍有 NaN

**原因**：数据太短或缺失过多

**解决**：
- 试试 `mean` 方法（用平均值填充）
- 或使用 `ffill` 方法（前向填充）

---

### ❌ 问题：Streamlit 启动很慢

**原因**：首次运行需要编译依赖

**解决**：
- 耐心等待（通常 30-60 秒）
- 之后重启会很快
