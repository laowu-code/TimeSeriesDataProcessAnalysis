## ✨ 核心功能速览

### 模块 1：📥 数据导入
- 上传 CSV/Excel 文件
- 自动识别时间和数值列
- 导出清洗后的数据

### 模块 2：📋 数据概览
- 查看数据统计信息
- 计算均值、中位数、标准差等
- 分析缺失值比例

### 模块 3：🔍 数据清洗
- 检测时间连续性
- 6 种插补算法可选
- 效果对比可视化

### 模块 4：📈 可视化
- 3 种图表类型
- Plotly 交互式图表
- 多变量对比

### 模块 5：⚠️ 异常检测
- 3 种检测算法
- 异常点自动标记
- 删除或修复选项

### 模块 6：🔬 高级分析
- STL 季节性分解
- 相关性热力图
- 多列趋势对比

## 📊 测试数据

项目包含 **sample_data.csv**（3 天气象数据），可直接使用测试所有功能。

该数据包含：
- ✅ 时间列（timestamp）
- ✅ 4 个数值列（temperature, humidity, pressure, pm25）
- ✅ 1 个缺失值（演示插补）

```
4. 📈 绘制时间序列图
5. ⚠️ 检测和处理异常值
6. 🔬 进行高级分析
```

## ⚡ 快速参考

### 问题排查

| 问题 | 解决方案 |
|------|--------|
| `ModuleNotFoundError` | 运行 `pip install -r requirements.txt` |
| 时间列未识别 | 确保格式为 `YYYY-MM-DD HH:MM:SS` |
| 启动慢 | 正常，首次需编译依赖 |
| 上传文件限制 | 编辑 `~/.streamlit/config.toml` 中的 `maxUploadSize` |

更多问题见 **README.md** 或 **QUICKSTART.md**

### 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+C` | 停止应用 |
| `R` | 刷新应用（在浏览器） |
| `C` | 清除缓存 |

## 💡 使用技巧

### 💾 导出数据
在 **📥 数据导入** 模块的导出选项中，支持：
- CSV 格式
- Excel 格式（.xlsx）

文件名自动包含时间戳

### 📊 优化图表显示
- **缩放**：在图上拖拽选区
- **平移**：按住鼠标拖动
- **查看数值**：鼠标悬停在数据点
- **重置**：双击图表

### 🎨 调整显示
编辑 `~/.streamlit/config.toml`：
```toml
[theme]
base = "light"  # 或 "dark"
primaryColor = "#FF6B6B"
```

## 🔄 工作流示例

### 场景：分析温度传感器故障数据

```
1️⃣ 上传包含缺失的温度数据
   → 📥 数据导入 → 上传文件

2️⃣ 查看数据概况  
   → 📋 数据概览 → 确认缺失率 25%

3️⃣ 检测和插补缺失值
   → 🔍 数据清洁 → 选择 Linear 插补 → 执行

4️⃣ 可视化原始 vs 插补数据
   → 📈 可视化分析 → 绘制图表

5️⃣ 识别异常高温点
   → ⚠️ 异常检测 → 用 IQR 方法 → 标记异常

6️⃣ 分析温度日周期
   → 🔬 高级分析 → STL 分解（周期=24）

7️⃣ 导出清洁数据供下游使用
   → 📥 数据导入 → 导出 CSV/Excel
```
